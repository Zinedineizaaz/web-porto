Machine Learning • Classification

Telco Customer Churn Prediction

Python Random Forest SMOTE Hyperparameter Tuning

Masalah Bisnis: Perusahaan telekomunikasi mengalami tingkat berhenti berlangganan (churn) yang tinggi. Model standar gagal mendeteksi pelanggan yang akan pergi karena ketidakseimbangan data (Imbalanced Data).

Solusi & Hasil: Saya membangun model End-to-End menggunakan Random Forest yang dioptimasi dengan teknik SMOTE dan Hyperparameter Tuning. Hasilnya, kemampuan mendeteksi pelanggan churn (Recall) meningkat drastis dari 46% menjadi 68%, berpotensi menyelamatkan ratusan pelanggan setiap bulannya.

Python NLTK TF-IDF Naive Bayes

Overview: Menganalisis ribuan ulasan film secara otomatis untuk menentukan sentimen positif atau negatif menggunakan teknik pemrosesan bahasa alami.

Python ETL PostgreSQL Star Schema OLAP Analysis
